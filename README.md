# School Shootings
 

I've showcased several Tableau Dashboards utilizing the data reported by The Washington Post on school shootings in America. Each Tableau Workbook can be located in the "Tableau Workbooks" directory within this repository. Please feel free to explore the dashboard images provided and download any workbook that catches your attention.
